# Code of conduct

While the following cannot be enforced, you are asked to respect it (and might be removed from and avoided in any relevant communities if you do not):

When making a project using Hakkun, do not:
- use LLMs to write or assist you in writing your code
- use any other form of generative AI in your project
- make software that encourages users to engage in behaviors that might get their switch consoles restricted from online services (such as: sending malformed or malicious requests to Nintendo's servers, allowing an unfair advantage to be enabled during an online game, or installing unlicensed titles to the console)
- make software that grabs a user's data with malicious intent (such as PRODINFO)
- make software (mainly mods for games) that merely exists to increase engagement and/or revenue for videos or media on YouTube and similar social media platforms, as opposed to existing to actually be useful or directly fun for end users
